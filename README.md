# CV_Project

主要包含三个部分：  
1.前端文件driving_agent_web：采用Vue搭建，主要包含以下页面：注册、登录、实时追踪、智能驾驶、历史告警、告警分析  


2.后端文件driving_agent：采用Python搭建，主要包含以下api：智能体deepseek-v4-flash模型api、前端功能api、数据库功能api  


3.数据库driving_agent_db：采用Mysql构建，主要包含两张表：用户表users、告警记录表alarm_records
